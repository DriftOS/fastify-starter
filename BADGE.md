## Get Featured

Using this starter in production? Get your project featured in the "Used By" section!

**How to submit:**
1. Fork [DriftOS/fastify-starter](https://github.com/DriftOS/fastify-starter)
2. Add your project to the "Used By" section in README.md
3. Submit a Pull Request
4. Include: Project name, URL, brief description

**Example:**
```markdown
- **[YourProject](https://yourproject.com)** - Brief description of what you built
```
